Массивы

Массивы представляют собой контейнеры объединённых вместе данных, в частности, данных, отправляемых для проработки из html-формы, как список товаров в корзине интернет-магазина. Массив представляет собой вместилище, вмещающим многие данные.

Управление массивами очень характерно для кодирования на PHP.

Массив включает в себя упорядоченные данные. У всех элементов массива имеются ключ и значение. В частности, массив, хранящий информацию о кассовом чеке, содержит названия продуктов и количество купленного товара.

Один из способов создания массива (array) – это конструкция array(). При создании она ему нужно передать данные которые массив буде содержать в виде ключа и значения (key => value).

 

array (

    индекс1   =>  значание1,

    индекс2   =>  значение2,

    индекс3   =>  значение3,

    ... и т.д.

)

После последнего элемента можно не ставить зарятую. Так создаются однострочные массивы, в частности, array (2, 3) предпочтительней array (1, 2, ). А если создается многострочный массив, то лучше оставить запятую после последнего элемента, потому что так легче добавлять новые элементы.

Что бы получить какое либо содержимое массива, нужно использовать словосочетание array [ key ].
