Методические рекомендации для преподавателя

Методические рекомендации для обучающегося (самостоятельная работа):

Преподавателю:

  • Организовать просмотр динамической инфографики и последующую беседу по уточнению и конкретизации первичных знаний.
  • Создать условия для освоения обучающимися знаний в форме интерактивной деятельности.
  • Формировать у обучающихся мотивацию к усвоению нового материала, образованию как основному инструменту
  • Достижения личного и профессионального успеха.
  • Познакомить обучающихся с основами разработки дизайн-концепции веб-приложений в соответствии с корпоративным стилем заказчика.
  • Раскрыть основы разработки дизайна веб-приложения с учетом современных тенденций в области веб-разработки.
  • Научить выбирать наиболее подходящее для целевого рынка дизайнерское решение.
  • Содействовать установлению в сознании обучающихся устойчивых связей между накопленным и новым опытом познавательной деятельности.
  • Фиксировать предложенные обучающимися гипотезы, курировать их обсуждение.

Студенту:

  • Ознакомьтесь с теоретическим материалом по новой теме. Просмотрите динамическую инфографику по теме «Технология проектирования и разработки интерфейсов пользователя».
  • Если в процессе изучения материала возникнут вопросы, запишите их для дальнейшего обсуждения с
  • Преподавателем и/или с группой обучающихся.
  • Обратите внимание на основные вопросы, подтверждающие взаимосвязь знаний и умений при изучении модуля:
    • Проектирование сайта.
    • Характеристики цвета.
    • Взаимодействие пользователя.
    • Юзабилити веб-сайтов и приложений для мобильных устройств.
  • При необходимости прослушайте/просмотрите материал еще раз.
  • Возвращайтесь к наиболее сложным аспектам темы.
  • Соблюдайте здоровье-сберегающий режим: чередуйте работу с электронными носителями с отдыхом и гимнастикой для глаз.
